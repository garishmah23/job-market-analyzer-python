# 📊 Job Market Analyzer

A beginner-friendly Data Analytics project built using Python, Tkinter, Pandas, and Matplotlib. This application helps users analyze job market trends, discover in-demand skills, explore salary insights, and visualize hiring patterns through an interactive desktop interface.

---

## 🚀 Features

* 📈 Analyze job market trends
* 🔍 Search jobs by skill
* 💰 View average and highest salaries
* 📊 Generate skill demand charts
* 📉 Salary analysis visualization
* 🏆 Display top-paying jobs
* 📋 View job data in tabular format
* 🖥️ User-friendly Tkinter GUI

---

## 🛠️ Technologies Used

* Python
* Tkinter
* Pandas
* Matplotlib
* NumPy

## 📦 Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/job-market-analyzer-python.git
cd job-market-analyzer-python
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python main.py
```

## 📊 Dataset

The project uses a sample CSV dataset containing:

| Column    | Description          |
| --------- | -------------------- |
| Job Title | Name of the job role |
| Company   | Hiring company       |
| Location  | Job location         |
| Skill     | Required skill       |
| Salary    | Offered salary       |
